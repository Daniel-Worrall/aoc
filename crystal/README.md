# AoC - Crystal

Advent of Code - Crystal

## Usage

`shards build` and run.

```
Usage: aoc [arguments]
    -a, --all                        Runs all solutions.
    -r NAME, --run=NAME              Specifies the code to run. Supports regex.
    -bm, --benchmark                 Benchmarks the solutions.
    -h, --help                       Show this help.
```
